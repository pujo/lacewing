Local (Filter)

C++
void Lacewing::Filter::Local (const char *)
C++
void lw_filter_set_local (lw_filter *, const char *)
C
Function not currently supported
JS
filter.local ([<Address or string>])

Description

Sets the local IP address associated with a Filter object.

This can be used to restrict a server-side class to listening on a particular IP address, when the system has more than one.

Parameters

Address

An Address object.

OR Hostname

Any string an Address object can resolve. May also contain a port number (eg. "192.168.1.2:80").
