Lacewing::Address

Description

Address is a general purpose class for representing the address of a remote host.

Example usages include describing the address of clients in the Server, the source and destination of datagrams in UDP, and the address to connect to in Client.

It is also possible to use Address independently as a utility for resolving a hostname.
