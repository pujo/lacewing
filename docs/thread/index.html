Lacewing::Thread

Description

Thread provides a simple, cross-platform facility for starting and joining a named thread.

The constructor for Thread takes two parameters :-

  • A pointer to the function
  • A user-defined name for the thread

The function may take a single pointer-sized parameter (as will be passed to Start) and return a 32-bit integer (returned by Join).

Remember that the majority of liblacewing is not thread-safe. Notable exceptions include EventPump::Post and EventPump::PostEventLoopExit.
