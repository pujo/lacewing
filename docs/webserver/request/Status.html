Status (Webserver::Request)

C++
void Lacewing::Webserver::Request::Status (int Code, const char * Message)
C
void lw_ws_req_set_redirect (lw_ws_req *, long code, const char * message)
JS
request.status (code, message)

Description

Sets the status code for the response (the default is 200 OK)

For a list of possible HTTP/1.1 status codes, see RFC 2616.

Parameters

Code

The numeric code of the status (eg. 200)

Message

A human readable message to further describe the status.
