LastModified (Webserver::Request)

C++
int Lacewing::Webserver::Request::LastModified ()
C++
void Lacewing::Webserver::Request::LastModified (lw_i64 Time)
C
long lw_ws_req_last_modified (lw_ws_req *)
C
void lw_ws_req_set_last_modified (lw_ws_req *, lw_i64 time)
JS
request.lastModified (<time>)

Description

Getter/setter for the last modified date of the requested data.

The date is represented in UNIX time (number of seconds since January 1st, 1970).

The getter returns the last modified date for the version of the resource the client has, or 0 if the client does not have a cached copy of the resource.

The setter sets the last modified date for the version of the resource sent as the response.
