Lacewing::Webserver::Request

Description

This class represents a single request to the Webserver. Each request issued by a client results in a new Request object being created (although Request objects may be reused internally, this should be ignored by the application).

The lifetime of a Request object as far as the application is concerned begins when it is passed to one of the handlers.
